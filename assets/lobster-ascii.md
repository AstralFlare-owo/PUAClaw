```
PUAClaw RFC 0099                                    The PUAClaw Consortium
Category: Visual Assets                             ISSN: 0000-0099
                                                    February 2026

        Official ASCII Art Lobster Collection — Standards
            and Usage Guidelines for Visual
                Crustacean Representation

Status of This Memo

   This document defines the canonical ASCII art lobster assets
   approved for use in PUAClaw publications. Unauthorized lobster
   representations are DISCOURAGED but not punishable (the lobster
   values artistic freedom). Reviewed by 147 lobsters, 3 of whom
   provided art direction.
```

## 1. The Simple Lobster (Compact Form)

```
  ,__
  |  `'.
  |_)  /
  |   (  🦞
  |    '-..___
  |           `"--._
  |                 \
```

**Usage**: Inline references, issue comments, quick acknowledgments. When you need a lobster but space is limited. Suitable for terminal output, CI logs, and casual reef banter.

**When to Use**: Everyday lobster needs. The equivalent of a friendly wave.

## Usage Guidelines Summary

| Lobster | Context | Formality Level | Minimum Occasion |
|---------|---------|-----------------|-----------------|
| Simple Lobster | Inline, comments | Casual | Any |
| Grand Lobster | Headers, banners | Formal | Document creation |
| Triumphant Lobster | Celebrations | Ceremonial | Merged PR, validated technique |
| Scholar Lobster | Academic contexts | Scholarly | Publication, review |
| PUAClaw Logo | Branding | Institutional | Official communications |
| Lobster Army | Mass consensus | Democratic | Votes, major releases |
| Lone Dissenter | Minority opinion | Informal | Dissent, skepticism |

### General Rules

1. Every major PUAClaw document SHOULD include at least one ASCII lobster, chosen appropriately from this collection.
2. ASCII lobsters MUST NOT be modified in ways that misrepresent the dignity of the species (sunglasses are acceptable; hats are debatable; mustaches require Ethics Board approval).
3. When in doubt about which lobster to use, default to the Simple Lobster. It is the Swiss Army knife of crustacean representation.
4. The Lobster Army MUST always total exactly 147 lobsters when fully deployed. Partial deployments are permitted with a note indicating how many lobsters remain in reserve.
5. Custom lobster art submissions are WELCOME and should be submitted via PR with the label `lobster-art`.
